Project Wikify

Summary

Project Wikify is a bookmarklet, a simple cross-browser browser-plugin that fully enables anyone to modify or append to any attribute of any website online, in a way that everyone else using the Wikify system can see and modify as they please. It forms a community on classicly static content. A method of exchanging data, ideas, and parody with ease. It enables for Web 2.0 to truly stretch across the vast width of the entire internet, not just a minority of select sites, truly upgrading the net.

Read-Only Wikify

Read-Only Wikify (or RWK) allows you to view the pages as they were intended with Project Wikify, but without write-access. It makes the pages feel more authentic, and Links and such actually work. It supports more browsers including Micosoft Internet Exploder 6+. To install, follow the install procedures for Project Wikify, but using the RWK link instead.

Information

Project Wikify is designed in such a way to only store user-contributed portions of the page, It enables for better support for somewhat dynamic pages, requires less space on the server, and (hopefully) saves me from some legal isues from doing this. It has no copies of the entire web site and never will (for all you overprotective, freedom-haters). As a purely community-based system, where anyone can do anythign without being segregated by logins, everything is entirely anonymous. So if you are dissatisfied on someone's contributions to a page, you can easily "turn the tables" on them.

Technical

Project Wikify is mostly in javascript, the server is very simple. It just stores and retrieves data.
