Intro

Imagine applying the collaborative wiki content model to all static content on the internet, allowing the community to exchange ideas, update content, fix errors, parody, and improve the internet as a whole.

Get It

Project Wikify is available as a cross-browser bookmarklet, which supports Mozilla Firefox, Opera, Apple Safari, Microsoft Internet Explorer, Google Chrome, and more. It provides all the tools necessary for editing, viewing, and saving.

A lightweight "Read-Only" version of Project Wikify is available as a userscript and as another bookmarklet.

Getting Started

Just install the bookmarklet, and start editing!
