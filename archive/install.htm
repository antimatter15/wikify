Project Wikify

How to install:

Drag the following link to your bookmarks toolbar.
Wikify

What it does:

Wikify turns any webpage into a Wiki, allowing you to improve the validity, and update outdated information on any site.

How it works:

When it is used, it turns on design mode for your browser. When you want to save, it checks for what changes you made, and only store your changes on the server. It has no copies of the entire web site and never will (for all you overprotective, freedom-haters), and anyone can revert anything. All changes are fully anonymous, no logging data is stored at all. In fact, all data is stored plaintext (on some server implementations).
