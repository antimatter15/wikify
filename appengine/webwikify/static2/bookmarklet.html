Install

Installing Project Wikify is as easily as bookmarking a web page.

On Mozilla Firefox, and Apple Safari, you can simply drag the Wikify link to your browser's bookmark toolbar. On Google Chrome, you must enable "Always Show Bookmarks Bar", and then you can follow the directions for Firefox. For Microsoft Internet Explorer, you can right-click on the Wikify link and select "Add to Favorites", and on Opera, you can right-click on the link and select "Bookmark Link".

Bookmarklet

A Bookmarklet is like a cross between a piece of software (plugin) and a simple link. It behaves somewhat like both, it's cross-browser, like a bookmark, and it does some plugin like things as well. When you click on a bookmarklet, instead of taking you to a page, it runs some small javascript code and modifies the existing page, in this case, it makes it editable to you and shows a tiny toolbar to control it.

Help

To use, simply run the bookmarklet (see install), and the Wikify toolbar should appear on the top right corner of your browser window. The page will automatically become editable as if it were a document in any word processor.

The Wikify toolbar includes a statusbar, "Save", "Update", "Toggle", "Switch", "Info" and "Help".

  • The save button submits your contributions to the patch server.
  • The update button reloads the current page, it is useful to check for whether or not another user has modified the page.
  • Toggle lets you switch between browsing and editing mode.
  • The switch button lets you enter a new "Channel" or a private branch of the internet. You can pick any of the main ones: Spam, Update, Talk, and Main, or you can enter any text.
  • The info button gives you general info on the current page, like revisions, etc.
  • Finally, the Help button gives you information about the version of project wikify, licensing, and basic usage
